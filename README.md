# Six's Quality Tweaks
Makes Quality modules just a little bit more powerful.

* Changes the effects of Mk 1/2/3 Quality modules from 1/2/2.5% to 1/3/5%
* This also results in, for example, a Legendary Mk3 Quality module having a 12.5% effect, up from 6.25%. A machine with 4 Legendary Mk3 Quality modules will have 50% quality.
* Higher tier labs have lower science pack drains, up to 50% at Legendary (Stacks with Biolab base bonus, so Legendary Biolab has 25% drain)
* Recyclers have a base +10% Quality Bonus. Use Speed modules to offset this when you want non-quality recycling for item deletion.
